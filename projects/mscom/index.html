MSCOM

Project Description

MSCOM is a complete front-end overhaul of Microsoft's hosting page. The project was generated with Yeoman and uses Grunt as a task runner to build and deploy to Microsoft Azure. Page animations and dynamic, templated social media content pulled in with jQuery.

Built With

  • HTML5
  • CSS3
  • jQuery
  • Node
  • Yeoman Generator
  • Grunt
